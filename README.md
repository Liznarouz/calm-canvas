# Calm Canvas — Launch-Ready React Website

A responsive React/Vite website for an art wellness business.

## Includes
- Home, Blog, Shop, Membership, About, Contact pages
- Responsive navigation
- Digital product cards
- Email lead magnet section
- SEO metadata
- Launch checklist
- Clean CSS design system

## How to run locally
```bash
npm install
npm run dev
```

## Build for launch
```bash
npm run build
```

## Deploy options
- Vercel: import this folder as a GitHub repo
- Netlify: drag and drop the `dist` folder after running `npm run build`
- GitHub Pages: use a Vite GitHub Pages workflow

## Replace before launch
- Add your real domain
- Replace email form action with Mailchimp, ConvertKit, or Beehiiv
- Replace checkout buttons with Stripe, Gumroad, Payhip, or Shopify links
- Replace placeholder brand copy if desired
